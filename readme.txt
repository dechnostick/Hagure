===============================================================================
　はぐれ計時
（フリーソフト）
================================================================================

■概要
　プログラムが起動するまでの時間を計るアプリケーションです。
　「はぐれけいじ」と読みます。

　何をもって起動の完了とするかはプログラムによってまちまちですが、
　このプログラムではプログラムの初期化をもって完了とみなします。

　＜開発者向けの説明＞
　プロセスの開始から WaitForInputIdle までを計ります。

　※　一部のアプリケーションはうまく時間を計れません。
　　　時間を計れない場合は「計測できませんでした。」と表示されます。

　※※　スプラッシュ(起動までの場つなぎで表示される画像等)のあるプログラムは、
　　　　実際のウィンドウが表示されるまでの時間よりも短い時間しか計測されない
　　　　場合があります。

■動作環境
　Windows Vista, Windows XP
　(Microsoft .NET Framework Version 2.0 が必要です)

■インストール方法
　はぐれ計時.exe を適当なフォルダに保存してください。

■アンインストール方法
　上記プログラムを削除して(ごみ箱へ入れて)ください。

■使い方
　１．はぐれ計時.exe を起動します。

　２．表示されたウィンドウに、時間を計りたいプログラム
　　　(またはファイル)をドロップします。

　３．かかった時間がウィンドウに表示されます。

　はぐれ計時.exe と同じフォルダに はぐれ計時.csv というファイルが作成されます。
　ここに今までに計った時間が記録されます。
　(計測日時、対象プログラム(ファイル)、所要時間が記録されます)
